## 2. 技术面高频考点

### 第一轮：Rust 基础 + 算法

必考题：
- 所有权/借用/生命周期（99% 出现）
- Send/Sync 的含义和推导规则
- async/await 原理和 tokio 运行时
- parking_lot vs std::sync（候选人项目中使用）
- LeetCode Medium 1-2 题

### 第二轮：项目深挖

面试官最可能追问的点：
1. "Clarity 和 LangChain/LlamaIndex 的核心差异？" → 本地优先、Contract-First 架构、零 Python 依赖、单二进制部署
2. "Agent 如何防止重复调用同一个工具？" → 步数限制 + 语义重复检测 + Token 预算
3. "为什么不用现成的 RAG 框架？" → 零外部向量数据库依赖、BM25+向量混合优于纯向量

### 第三轮：系统设计

高频 Agent 设计题：
- 设计一个多 Agent 协作的代码审查系统
- 设计一个支持 MCP 的工具网关
- 设计一个 RAG 知识库系统

### 第四轮：主管面

主管关心的：技术判断力、架构品味、学习能力、团队适配度。

## 3. 算法题备考策略

### Rust 刷题技巧

```rust
// HashMap entry API
map.entry(key).or_insert(0) += 1;

// VecDeque 做 BFS
let mut queue = VecDeque::new();
queue.push_back(root);

// BinaryHeap 做 Top-K（默认最大堆，Reverse 变最小堆）
let mut heap = BinaryHeap::new();
heap.push(Reverse(item));
```

### 推荐刷题清单（50 题优先级）

**高频必刷**（20 题）：两数之和、有效括号、合并两个有序链表、二叉树层序遍历、二分查找、岛屿数量、全排列、最长回文子串、LRU 缓存、三数之和

**重点练习**（20 题）：接雨水、最小栈、翻转链表、子集、爬楼梯、买卖股票、最长上升子序列、合并区间、二叉树最大路径和、课程表

**选做拔高**（10 题）：编辑距离、跳跃游戏、最长有效括号、滑动窗口最大值、寻找两个正序数组中位数

## 4. 行为面试（BQ）准备

### "为什么选择本科就业而非读研？"

> "AI 行业变化极快，2024-2026 这三年出现的东西比之前十年都多。我认为在这个领域，动手实践比学术培养更能跟上节奏。我的三个项目——Clarity 从设计到 1554 个测试——证明了我独立交付复杂系统的能力已经超过大多数研究生。读研能带来的深度，我可以在一线工作中通过解决真实问题来获得。"

### "你最大的弱点是什么？"

> "我缺乏大规模生产环境经验——没有经历过正式的 code review 流程、生产环境运维、on-call 轮值、跨团队协作。我通过在自己的项目中强制执行 CI/CD、测试覆盖、架构文档来部分弥补，但团队的动态和系统的规模是个人项目无法模拟的。这也是我为什么想加入一个有经验的团队——我想学习这些。"

### "为什么没有实习经历？"

> "我的时间用来独立构建大型开源项目——Clarity 22 个 crate、Devbase 71 个 MCP 工具、Syncthing-rust 392 个测试。这些项目让我从架构设计到 CI/CD 全流程独立负责。当然我很清楚独立开发和团队开发的差异——code review、跨职能协作、生产环境的灰度发布——这些是我下一步最想学习的。"

### "你如何处理技术分歧？"

> "如果分歧是技术性的，我倾向于数据驱动——用 benchmark 或 prototype 来验证。如果分歧是风格性的（缩进 vs 命名），我倾向于跟随团队惯例。如果面对更资深的工程师的不同意见，我会先假设对方有我没看到的信息，多问'为什么'来理解而不是直接反对。"

### "你未来 3-5 年的职业规划？"

> "短期（1-2年）：深入 AI Agent 基础设施方向，成为团队中 Agent 架构方面的 go-to person。中期（3-5年）：能够设计更大规模的 Agent 系统（百万 DAU 级别），从个人贡献者成长为技术骨干或架构师。长期：我希望成为 AI 基础设施领域的专家——不一定是最懂 AI 模型的人，但是最懂如何让 AI 可靠地工作的人。"

## 7. Offer 谈判与评估

### Offer 比较维度（按权重排序）
1. **成长空间**（40%）：技术栈匹配度、mentor 质量、项目影响力
2. **团队氛围**（25%）：技术文化、是否鼓励开源、WLB
3. **薪资**（20%）：基本工资 >> 年终奖 >> 期权（实习生/应届期权大概率不值钱）
4. **城市**（10%）：北上深杭优先
5. **公司品牌**（5%）：对简历的长期价值

### 薪资参考（2026年，一线城市，Rust/AI 方向）
- **实习**：8K-20K/月（字节/深度求索/月之暗面 15K+）
- **应届正式**：15K-35K/月 × 14-16薪
- **选项/股票**：实习生基本没有，应届偶有（上市公司 RSU，创业公司期权）
