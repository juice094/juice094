# 周景潇 — 岗位匹配与投递建议

## 匹配总览

| 匹配度 | 方向 | 核心依据 |
|--------|------|----------|
| ⭐⭐⭐⭐⭐ | **AI Agent / LLM 应用开发** | Clarity 14-crate Agent 框架 + MCP 三层传输，业内稀缺 |
| ⭐⭐⭐⭐⭐ | **Rust 后端 / 基础设施** | 三个 Rust 项目累计 2000+ 测试，工业级工程素养 |
| ⭐⭐⭐⭐ | **大数据 / 数据工程** | 专业对口 + Hadoop/Spark/Kafka + 均分前10% |
| ⭐⭐⭐⭐ | **开发者工具 / 平台工程** | devbase 71 MCP tools + CI/CD + Workflow 编排 |
| ⭐⭐⭐ | **P2P / 分布式系统** | syncthing-rust BEP 协议栈 + NAT 穿透实战 |
| ⭐⭐⭐ | **Golang 后端** | 技术栈覆盖，但 Rust 强信号更突出 |

---

## 方向一：AI Agent / LLM 应用开发 ⭐⭐⭐⭐⭐

**为什么匹配：**
Clarity 不是玩具项目——14 crate 拆分、四层 Approval 审批链、ReAct/Plan 双模式、MCP 三传输。这些在 2026 年 AI 招聘市场上是极度稀缺的实践信号。大多数候选人的"AI 项目"是调 API，宿的是造框架。

**推荐岗位关键词：**
- AI 应用开发工程师
- Agent 框架开发
- LLM 应用开发
- MCP 工具链开发
- AI-Native 产品工程师

**目标公司类型：**
- AI 中间件/基础设施创业公司（如 Dify、Coze 生态企业）
- 大厂 AI 平台部门（Agent 框架/工具链方向）
- 开源商业化公司（Rust + AI = 稀缺组合）

**简历话术建议：**
面试时把 Clarity 定位为"探索 AI Agent 工程化边界"而非"个人练手"。强调架构约束（contract 层零依赖、core 零前端依赖）——这在大厂面试中是加分级表述。

---

## 方向二：Rust 后端 / 基础设施 ⭐⭐⭐⭐⭐

**为什么匹配：**
三个 Rust 项目，2000+ 测试，clippy 0 warning，零 unwrap/expect。这不是"学过 Rust"，这是"用 Rust 做过工业级系统"。Rust 岗位在国内虽然总量不大，但供给更少，供需比极有利。

**推荐岗位关键词：**
- Rust 后端开发
- 分布式存储/数据库内核
- 基础设施工程师
- 系统软件工程师
- 云原生开发（Rust 方向）

**目标公司类型：**
- 数据库/中间件厂商（TiDB、RisingWave、GreptimeDB 等）
- 云厂商基础设施团队
- Web3/区块链（Rust 是主力语言）
- 量化交易系统

**注意：**
Rust 岗位通常要求较强 CS 基础（OS/网络/分布式），宿的项目经历覆盖了这些——BEP 协议栈 + NAT 穿透就是最好的证明。

---

## 方向三：大数据 / 数据工程 ⭐⭐⭐⭐

**为什么匹配：**
专业对口 + GPA 前 10% + Hadoop/Spark/Kafka 课程实践 + student-era 项目归档。作为大三学生，技术栈覆盖广度够用。但相比 Rust 的三个项目，大数据方向没有独立的口碑级产出。

**推荐岗位关键词：**
- 大数据开发工程师
- 数据平台开发
- 数据工程实习生
- ETL/数据仓库

**目标公司类型：**
- 互联网大厂数据平台部门（字节、美团、阿里）
- 银行/金融科技（工行软开、招行等——已有申请材料）
- 传统企业数字化转型团队

**策略：**
如果投工行，大数据方向最安全——学历对国企够用，项目经历是加分项。工行申请材料已备好（`icbc-2026-internship-application.md`）。

---

## 方向四：开发者工具 / 平台工程 ⭐⭐⭐⭐

**为什么匹配：**
devbase 71 个 MCP 工具 + Workflow YAML 编排 + CI/CD 门禁 + tree-sitter 多语言解析。这本质上是平台工程（Platform Engineering）的实践——为开发者构建工具链和工作流。

**推荐岗位关键词：**
- 平台工程
- 开发者工具开发
- DevOps / SRE（工具链方向）
- 工程效能

**目标公司类型：**
- 有内部工具平台团队的互联网公司
- 开发者工具创业公司
- 开源商业化公司

## 投递策略建议

1. **内推优先** — 叔叔那条线继续推进，绕过平台筛选效率最高
2. **双线并行** — 一份简历同时投 Rust/AI 方向和大数据方向，但需要微调侧重点：
   - AI/Rust 方向：项目经历放前面，强调框架设计能力
   - 大数据方向：教育背景+GPA前置，项目作为工程素养佐证
3. **作品即简历** — GitHub 三个仓库的 README/CONTRIBUTING/CI badge 保持整洁，面试官会点进去看
4. **时间窗口** — 6月是秋招提前批窗口，大三下的实习现在投正合适
5. **工行保底** — 材料已有，建议同步投递，国企流程慢需要提前启动
